# 부동산 매물 검색 MCP 서버

국토교통부 공공 API와 카카오맵 API를 활용한 부동산 실거래가 기반 매물 검색 MCP 서버입니다.

## 주요 기능

### 1. search_property - 매물 검색
지역, 매물 종류, 거래 유형, 가격 범위로 실거래 매물을 검색합니다.
- 아파트 / 오피스텔 / 빌라(연립다세대)
- 매매 / 전세 / 월세
- 동 단위 필터링, 역 근처 거리 기반 정렬
- 살기좋은순 정렬 (역세권, 학군, 신축 종합 평가)
- 주변 지하철역, 학교, 어린이집 정보 + 도보 시간
- 카카오맵 링크 제공

### 2. find_midpoint_property - 중간 지점 매물 추천
두 직장/학교의 중간 지점에서 최적의 매물을 추천합니다.
- 카카오맵 API로 정확한 중간 지점 계산
- 중간 지점 행정구역의 실거래 매물 조회
- 각 지점까지의 거리(km) 표시

### 3. get_market_price - 시세 조회
특정 아파트의 실거래가 시세를 조회합니다.
- 평형별 평균가, 최저가, 최고가
- 매매 + 전월세 동시 조회
- 최근 거래 내역 (건물명, 주소, 가격, 거래일)
- 카카오맵에서 아파트 위치 자동 검색

## 사용된 API

| API | 용도 | 비용 |
|-----|------|------|
| 국토교통부 아파트 매매 실거래가 | 매매 거래 데이터 | 무료 |
| 국토교통부 아파트 전월세 실거래가 | 전월세 거래 데이터 | 무료 |
| 국토교통부 오피스텔 매매/전월세 | 오피스텔 데이터 | 무료 |
| 국토교통부 연립다세대 매매/전월세 | 빌라 데이터 | 무료 |
| 카카오맵 키워드 검색 | 장소 좌표 검색 | 무료 |
| 카카오맵 카테고리 검색 | 주변 역/학교/어린이집 | 무료 |
| 카카오맵 좌표→행정구역 | 지역 특정 | 무료 |

## 기술 스택

- Python 3.12
- MCP SDK (FastMCP)
- httpx (비동기 HTTP 클라이언트)
- Docker
- 카카오클라우드 배포

## 프로젝트 구조

```
kakao-real-estate-mcp/
├── pyproject.toml
├── requirements.txt
├── Dockerfile
├── .env.example
└── src/kakao_real_estate/
    ├── __init__.py
    ├── server.py          # MCP 서버 (3개 도구)
    ├── api_client.py      # 국토교통부 + 카카오맵 API 클라이언트
    └── region_code.py     # 법정동 코드 + 장소 좌표 매핑
```

## 설치 및 실행

### 환경변수 설정
```bash
cp .env.example .env
# .env 파일에 API 키 입력
# DATA_GO_KR_API_KEY=공공데이터포털_키
# KAKAO_REST_API_KEY=카카오_REST_API_키
```

### 로컬 실행
```bash
pip install -e .
python -m kakao_real_estate.server
```

### Docker 실행
```bash
docker build -t kakao-real-estate-mcp .
docker run -p 8000:8000 kakao-real-estate-mcp
```

### MCP 엔드포인트
```
http://localhost:8000/mcp
```
